# ozip2zip
OPPO firmware ozip to zip

Library
---

```pip install pycrypto```

Prebuilt pycrypto Binaries for Windows download here: http://www.voidspace.org.uk/python/modules.shtml#pycrypto

Usage
---

```ozip2zip.py <*.ozip>```

Tested devices
---

```
OPPO R9s
OPPO R9s plus
OPPO R11
OPPO R11s
OPPO R11s plus
```
